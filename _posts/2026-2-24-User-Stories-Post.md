Hello, this post will be about my user stories that I created for my Lab 6 assignment. These are the five user stories that I came up with:
<ol>
<li>As a Customer, I can register at the website so that I can place orders for pick up or delivery.</li>

<li>As an Owner, I can list items for sale so that Customers can select the items that are being sold.</li>

<li>As an Employee, I can view the Customers finalized order so that I can fulfill their order.</li>

<li>As a Customer, I can choose the time and date that my order will be fulfilled so that an employee can accurately fulfill it.</li>

<li>As an Owner, I can view the inventory on items so that I can relay to the Employees what needs to be restocked.</li> 
</ol>
